# Coursework: РАЗРАБОТКА ПРИЛОЖЕНИЙ И DIRECTSHOW-ФИЛЬТРА ДЛЯ ПЕРЕДАЧИ ВИДЕОПОТОКА С КАМЕРЫ МОБИЛЬНОГО УСТРОЙСТВА В ПРИЛОЖЕНИЯ НА WINDOWS ЧЕРЕЗ WI-FI

## Проекты:
- `Desktop/` — Серверное приложение для Windows (.NET, WPF)
- `Phone/` — Приложение для Android (Kotlin, CameraX)
- `VirtualCamFilter/` — Виртуальная камера (C++, DirectShow)
- `vCamTest/` — Использовал для тестрования работы виртуальной камеры на ранних этапах
- `coursework` — Папка с PDF-файлом полного текста курсовой работы

## Краткая инструкция

### Как собрать и запустить:
1. Откройте `Desktop/CameraReceiver.sln` в Visual Studio. Соберите и запустите как администратор. Либо можно через publish.but, .exe будет в папке publish-output
2. Откройте проект в `Phone/` через Android Studio, соберите и установите на телефон.


### Краткое описание

- Сервер на Windows принимает кадры по Wi-Fi и направляет их в виртуальную камеру.
- Android-приложение отправляет кадры с камеры устройства.
- Виртуальная камера доступна в любых приложениях под Windows.
